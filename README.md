**Smart Home Automation System**

This project is a Smart Home Automation System that allows users to monitor and control devices across different rooms through a centralized dashboard. Users can manage devices, create automations, and view energy usage analytics to understand overall consumption patterns.
The system includes both frontend and backend components to simulate real-time device control and provide insights for efficient home management.

**Technologies Used**:
* React (Frontend)
* MongoDb (Backend)
* Bootstrap (UI Styling)

**Features**
* Room and device management
* Automation controls
* Energy usage tracking and analytics
* User-friendly dashboard interface
